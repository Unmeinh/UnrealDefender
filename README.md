- What is this?
- This is a project game I made, names UnrealDefender. The game base on pvz's tower defend type, but on another dimension when humans are occupying the fantasy lands of other races, and the Dark Elf Queen's castle is the final line of defense of the fantasy realm. Player will assist the Dark Elf Queen in defending her castle against the human onslaught.

# How to play
 
## Step 1: Go to the pre-release page

Go to this page [pre-release](https://github.com/Unmeinh/UnrealDefender/releases/tag/demo) to get the pre-release file of the game.

## Step 2: Download the pre-release.zip on the lastest demo
>**Note**: Its only have demo 1.0 for now so just download the file from it.

## Step 3: Unzip the pre-release.zip
>**Note**: You know how to do it so... no guide here.

## Step 4: Run the UnrealDefender.exe to start playing.
>**Note**: Thank you very much for playing.
